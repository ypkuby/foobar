Editors / IDEs

Tere's tons of editors and IDEs out there. Here are some easy to install and configure editors. Some editors may require payment, they will have an asterisk (*) next to their name.


  • Microsoft Visual Studio Code

    A great IDE, very popular amongst developers. Easy to customize, fast to start, and no steep learning curve.

  • Atom by Github

    A hackable text editor, created by Github

  • Jetbrains IDEs*

    Very popular with the development community, common IDEs include PyCharms (Python), PHPStorm (PHP), WebStorm (HTML/CSS/JS/TS/NODE), GoLand (Golang), RubyMine (Ruby), DataGrip (databases and SQL) and more. These require payment, however, are available to students for free and open-source projects meeting a criteria.

  • Sublime Text

    Very fast editor, can be turned into a full-blown IDE with plugins. Automatic saving, quick to open, and incredibly fast to edit files.

  • Notepad++

    A classic multi-purpose editor that gets the job done. Great for when you need a lightweight editor.
